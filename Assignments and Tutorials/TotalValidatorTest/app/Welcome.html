Welcome

Thank you for selecting Total Validator as your website testing tool.

To access all the features of Total Validator, such as Document Object Model (DOM) testing for javascripted sites, we recommend that you install our browser extensions.

To ensure that you get the best out of our tool, please use the 'Documentation' link above to access the documentation. This includes a quick start guide, comprehensive explanations of all the options, a few How-To guides, and an FAQ covering common issues experienced by first time users. This help may also be accessed when you are running Total Validator by pressing the F1 key.

If you can't find what you need in the documentation then please use the Contact Form on our website to ask for help.

Updates

Total Validator is updated several times a year with new features, changes to the standards, and bug fixes. Please see the change log for a list of the latest updates.

You can check for updates at any time, or automatically each time you start Total Validator, using the Options menu.
